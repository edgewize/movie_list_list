
# Install
```
$ git clone https://github.com/edgewize/movie_list_list.git
$ cd movie_list_list
$ pipenv install
$ npm install
$ cd node_modules/fomantic-ui
# output goes to project/static
$ npx gulp install
$ npx gulp build
$ pipenv shell
$ python build.py
$ flask run
```

# For local dev

create .env file
```
FLASK_APP= project
FLASK_DEBUG= < 1 >
TMDB_API_KEY= < your key>
```
Run project
```
$ pipenv shell
$ flask run
```
